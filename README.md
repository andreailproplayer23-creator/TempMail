# ⚡ TempMail - Fast Temporary Email Manager

TempMail è un'applicazione desktop leggera e potente sviluppata in Python per la gestione di email temporanee multiple. 
Grazie all'integrazione con l'API di Mail.tm, permette di generare istantaneamente fino a 10 indirizzi email contemporaneamente con notifiche desktop in tempo reale.

---

## ✨ Caratteristiche
* **Generazione One-Click**: Crea una nuova email istantaneamente con un solo tasto.
* **Multi-Account**: Gestisci fino a 10 sessioni contemporaneamente in tab separati.
* **Supporto Multilingua**: Interfaccia disponibile in Italiano, English, Español e Français.
* **Persistenza**: Le sessioni attive vengono salvate localmente e ripristinate all'avvio.
* **Notifiche Real-time**: Ricevi avvisi direttamente su Windows all'arrivo di nuovi messaggi.
* **Timer Integrato**: Visualizza la scadenza di ogni casella postale in tempo reale.

---

* Python 3.10+
* CustomTkinter
* Requests
* Win10Toast
* Pyperclip

## 🚀 Installazione
Per gli utenti Windows, è disponibile un comodo installer nella sezione [Releases](https://github.com/andreailproplayer23-creator/CaveauDigitale/releases).
1. Scarica `TempMail_Setup.exe`.
2. Esegui l'installer e segui le istruzioni.
3. Avvia TempMail dal desktop o dal menu Start.
